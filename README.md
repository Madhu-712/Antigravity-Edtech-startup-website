## üìñ About The Project

**Edu-genius-Tech Tutors** is a premium EdTech platform designed to bridge the gap between ambitious students and world-class educators. The platform focuses on high-demand technical fields like AI, Machine Learning, Data Science, and IoT, while also catering to K-12 education and exam preparation.

### Key Features
*   **Smart Tutor Search**: Find experts by subject, price, rating, and experience.
*   **Interactive Booking**: Real-time calendar integration for scheduling sessions.
*   **User Dashboards**: Dedicated portals for Students (progress tracking) and Tutors (business management).
*   **Educational Resources**: Access to Mock Interviews, Project Libraries, and Question Papers.
*   **AI-Powered Support**: Integrated chatbot for instant query resolution.

## üèÅ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites
*   Node.js (v18 or higher)
*   npm or yarn

### Cloning the Repository

```bash
git clone https://github.com/Madhu-712/Antigravity-Edtech-startup-website.git
cd edu-genius-tech-tutors
```

### Installation

Install the project dependencies:

```bash
npm install
# or
yarn install
```

### Running the Application

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) (or the port shown in your terminal, usually 3002 if 3000 is busy) in your browser to see the application.
